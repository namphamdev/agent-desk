How music variant generation progress is shown

There's a layered system. "Progress" is actually several distinct signals, each rendered differently. Here's how each variant shows its progress while generating.

What you see on each variant card

The per-variant UI lives in VariantCard inside src/screens/dashboard/library/components/VariantSelector.tsx. Each card shows up to six things depending on its stage:

Visual When it shows Code
Numeric % badge Only when variant.stage === 'generation' VariantS
Skeleton pulse for the title Title not yet streamed in ( variant.pending ) VariantS
Skeleton pulse for tags Display tags pending VariantS
Animated EQ-bars icon Active variant + playing EQBars
Playback progress bar (thin, bottom) Active variant while listening — not generation VariantS
