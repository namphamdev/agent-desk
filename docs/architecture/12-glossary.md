# 12. Glossary

| Term | Meaning |
| --- | --- |
| **ACP** | Agent Client Protocol — stdio JSON-RPC between host and coding agent |
| **ADR** | Architecture Decision Record under `09-architecture-decisions/` |
| **arc42** | Architecture documentation template used in `docs/architecture/` |
| **Harness** | Project optimizations applied by TR (e.g. AGENTS.md, skills) |
| **Hot fact** | Short, always-relevant bullet kept in `docs/memory/INDEX.md` |
| **INDEX** | Always-loaded memory catalog (`docs/memory/INDEX.md`) |
| **Journal** | Raw, not always-loaded capture under `docs/memory/journal/` |
| **Learning loop** | Capture → promote → skill/architecture update → git sync |
| **Promote** | Move durable journal content into topic files; trim journal |
| **Skill** | `SKILL.md` package — procedural knowledge loaded on demand |
| **Topic file** | Sharded durable memory under `docs/memory/topics/` |
| **TR** | terminal-react (this application) |
