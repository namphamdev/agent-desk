## 2026-07-13 — Windows enlarge click-through
- Context: After resizing the desktop app larger on Windows, clicks in the new region went to the window behind.
- Lesson: Electrobun transparent/layered windows + WebView2 lag leave new client area non-hittable. Disable `transparent` on win32; keep `hiddenInset` for resizable custom chrome.
- Promote to: topics/incidents.md (done)

## 2026-07-13 — POSIX-only shellouts (pgrep/ps) break on Windows
- Context: Memory RSS sampling shelled out to `pgrep -P` + `ps`, which don't exist on win32 → ENOENT every 2s poll (`[acp] memory sample failed`).
- Lesson: Any `spawn(cmd, ...)` of a Unix tool needs a `process.platform === "win32"` branch. For process-tree work, use PowerShell: `Get-CimInstance Win32_Process -Filter 'ParentProcessId=<pid>'` for children, `Get-Process -Id <pids> | Measure-Object WorkingSet64 -Sum` for RSS.
- Promote to: topics/incidents.md (done)

## 2026-07-20 — Stop memory hook was optional echo only
- Context: Project-memory harness only wrote `.claude/settings.example.json` with a command `echo` on Stop; real `.claude/settings.json` was never installed, so memory often stayed stale after agent turns.
- Lesson: Scaffold a **prompt-type** Stop gate into project `.claude/settings.json` (merge, upgrade legacy echo, keep other hooks). Gate blocks stop only when a team-durable lesson exists and is not yet journaled; prefer approve when unsure. Harness apply is the install path.
- Promote to: topics/tooling.md | topics/incidents.md | discard further detail
